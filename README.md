## Installation

```
$ git clone https://github.com/Karamysh/php-project-45.git

$ cd php-project-45

$ make install

```

## Start games:

```
$ make brain-games

$ make brain-even

$ make brain-calc

$ make brain-gcd

$ make brain-progression

$ make brain-prime

```
